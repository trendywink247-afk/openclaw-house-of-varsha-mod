# n8n Automation Builder Skill

This skill automates the process of creating, deploying, and managing n8n workflows, with a focus on services like Instagram and ManyChat, and deployment via MCP Server.

## Features

- Design workflows triggered by social media events
- Connect and integrate APIs like Instagram and ManyChat
- Programmatically deploy workflows using n8n MCP Server API
- Optimize automation for cost efficiency

## Usage

1. Define the trigger events and action workflows.
2. Map keywords or inputs to specific actions or data responses.
3. Use built-in functions to parse inputs and generate outputs.
4. Deploy and manage the workflows programmatically via the MCP Server API.

## Example

The Instagram comment to ManyChat DM automation:

- Monitor Instagram comments for specific keywords
- Parse and map keywords to product information
- Trigger ManyChat DMs with product cost or dialog flow if user is not following
